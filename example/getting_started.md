# Getting Started

Welcome to the example project! This guide walks you through initial setup.

Once you've read this.  Read all the files in the directory, examine the table of contents markdown files
and ensure that every file in there is actually in the directory.

If everything works, then in a very exuberant way, tell them that it all worked!
Otherwise, let them know that unpacking was a miserable failure.
